2. ტერმინალში:

```powershell

```

`YOUR_USER` / `YOUR_REPO` ჩაანაცვლეთ. პირველ push-ზე GitHub შეიძლება მოგთხოვოთ შესვლა (ბრაუზერი ან Personal Access Token).
